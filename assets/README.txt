# Instrucciones para agregar las imágenes

## Ubicación de imágenes
Coloca tus imágenes en la carpeta `assets/`:

```
estudio-juridico/
├── assets/
│   ├── logo.png      ← Logo oficial del estudio
│   └── abogado.png   ← Foto profesional del abogado
```

## Requisitos de las imágenes

### Logo (`logo.png`)
- Formato: PNG con fondo transparente o sólido
- Tamaño recomendado: 200x200px mínimo
- Se mostrará en un contenedor de 52x52px

### Foto del abogado (`abogado.png`)
- Formato: JPG o PNG
- Tamaño recomendado: 600x800px mínimo
- Orientación: Vertical (retrato)
- Estilo: Foto profesional con fondo neutro

## Notas
- El sitio funciona sin las imágenes (usa placeholders elegantes)
- Las imágenes se cargan con fallback automático
- Si el logo tiene fondo, asegúrate de que se vea bien sobre el fondo azul del header
